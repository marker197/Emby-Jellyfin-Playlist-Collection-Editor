## **Emby Chronological Playlist Generator — Complete Feature Breakdown**

**What it is:** A single-file HTML app that creates perfectly-ordered movie playlists on your Emby server. No installation, no cloud service, runs entirely in your browser.

### **🎬 Core Features**
- **100+ Built-in Franchises** — Marvel, Star Wars, James Bond, Harry Potter, etc. Pre-loaded with correct chronological order
- **Create Playlists** — Match franchise movies against your library and auto-order them (Release Date or Production Order)
- **Refresh Existing Playlists** — Re-sort and auto-add newly downloaded movies
- **Preview Before Creating** — See exactly which of your movies match before writing anything to Emby
- **Bulk Mode** — Create 50+ playlists in one batch
- **Collections Mode** — Create Emby collections instead of playlists for better metadata preservation

### **⭐ Franchise Management**
- **Custom Franchises** — Add your own movie series
- **CSV Import** — Bulk-upload custom franchise lists
- **TMDB Enrichment** — Auto-update franchises from The Movie Database with one click
- **Source Toggle** — Switch between Built-in, Custom, and TMDB ordering per playlist
- **Conflict Resolution** — Choose how to handle duplicate franchises across sources

### **🎯 Smart Matching**
- **Precision Title Matching** — Prevents false positives (won't match "RV" to "Marvel")
- **Year-aware Matching** — Handles remakes and regional title variants
- **Fuzzy Matching** — Handles minor title variations

- **Radarr Integration** — Request missing movies directly from the app

- **Trakt Lists** — Import your Trakt watchlists and create collections from them
- **Auto-Reconnect** — Radarr servers auto-reconnect if they drop
